# AtlasRyb_data-base-sql
Nazwa projektu : Atlas ryb
Opis: Jest to aplikacja zawierająca informacje o rybach, ochronie poszczególnych gatunków,
przynętach i łowiskach. W aplikacji można znaleźć też informacje na temat Okręgów
Polskiego Związku Wędkarskiego, członkach zarządu oraz o cenach kart rybackich w
poszczególnych okręgach
Cel i zakres pracy: Funkcjonalność aplikacji powinna obejmować:
 Wyszukiwanie informacji o rybach oraz ochronie poszczególnych gatunków
 Wyszukiwanie informacji o łowiskach oraz o tym jakie gatunki w nich występują
 Wyszukiwanie informacji
